$$

$$

$$


1. $\overline{\overline{Y}} = Y$  
2. $\overline{X} \rightarrow Y \equiv X \lor Y$  
3. $(X \lor Y) \lor Z \equiv X \lor Y \lor Z$  
4. $(\overline{X} \land (Y \lor Z)) \lor (X \lor Y \lor Z) \equiv X \lor Y \lor Z$
